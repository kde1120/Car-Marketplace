## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## 설치 및 실행

1. 프로젝트 클론:

   ```
   git clone [repository-url]
   cd vehicle-management-system
   ```

2. 의존성 설치:

   ```
   npm install
   ```

3. 애플리케이션 실행:
   ```
   npm run start
   ```

## API 문서

애플리케이션이 실행된 후, Swagger UI를 통해 API 문서를 확인할 수 있습니다:
http://localhost:3000/api
